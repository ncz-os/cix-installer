# nclawzero branding assets

Plymouth theme assets, login wallpapers, GDM logo, etc. are dropped here
and consumed by `post-install/60-plymouth.sh` and `50-brand.sh`.

v0 just uses Debian's default `spinner` Plymouth theme (already polished
enough to look intentional). v1 will add a custom theme with the
nclawzero wordmark + edge-agent visual language.
