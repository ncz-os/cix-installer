# server-seed.txt — the NCZ "Ubuntu Server library" seed.
#
# Policy (2026-06-16): NCZ maintains a Server-class base offline; end-user /
# desktop packages are pathed out to Ubuntu (ports.ubuntu.com) online. This
# file is the explicit server seed layered ON TOP OF every package of
# Priority required/important/standard (which build-server-mirror.sh adds
# automatically). The dependency closure of all of these = the offline
# server mirror. Anything not here and not a dependency = dropped (desktop).
#
